# Samer Yousry Portfolio

A personal portfolio website built with Next.js to showcase projects, skills, experience, and contact information.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Radix UI

## Getting Started

### 1) Install dependencies

Using npm:

```bash
npm install
```

Or using pnpm:

```bash
pnpm install
```

### 2) Run development server

Using npm:

```bash
npm run dev
```

Or using pnpm:

```bash
pnpm dev
```

Open http://localhost:3000

## Available Scripts

- `dev`: Start local development server
- `build`: Build for production
- `start`: Start production server
- `lint`: Run lint checks

## Project Structure

- `app/` - App Router pages and layout
- `components/` - Portfolio sections and UI components
- `public/` - Static assets (logo, icons, images)

## Deployment

This project is optimized for Vercel deployment.

It can also be deployed on Netlify, but Vercel is typically the smoother option for Next.js projects.
